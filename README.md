#基于stm32f103c8t6的ADC多通道测量传感器模拟输出

##硬件说明
**主控芯片**:STM32f103c8t6
**传感器**:光敏传感器;温度传感器;红外对射式传感器

##功能演示
-实时采集以上三种传感器的实时输出的模拟信号
-通过OLED小屏实时展示模拟信号的数值大小(0~4096)
